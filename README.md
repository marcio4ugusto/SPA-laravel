# Laravel SPA

A Monolith SPA scaffolding with Laravel, Inertia JS/React and Tailwind.

## Use

```shell
  git clone git@github.com:maaugusto/SPA-laravel.git
  cd SPA-laravel
  composer install
  php artisan key:generate
  npm install
  npm run dev
  php artisan serve
```

Feel free to use it as a start point for your projects.
Cheers!
